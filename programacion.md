···1. ¿Cuáles son los tipos de datos primitivos en Java?

Los tipos de datos primitivos en Java son ocho:

- **byte:** números enteros pequeños.

- **short:** números enteros cortos.

- **int:** números enteros comunes.

- **long:** números enteros grandes.

- **float:** números decimales de precisión simple.

- **double:** números decimales de mayor precisión.

- **char:** almacena un solo carácter.

- **boolean:** almacena valores verdadero o falso (true o false).

···2. ¿Cómo funcionan las estructuras de control de flujo como if, else, switch y bucles en Java?

Las estructuras de control permiten decidir el orden en que se ejecuta el programa.

- **if:** ejecuta un bloque de código si una condición es verdadera.

- **else:** ejecuta otro bloque cuando la condición es falsa.

- **switch:** selecciona una acción dependiendo del valor de una variable.

- **Bucles (for, while, do-while):** repiten instrucciones varias veces mientras se cumpla una condición.

 ···3. ¿Por qué es importante usar nombres significativos para variables y métodos?

Porque facilitan la lectura y comprensión del código. Un nombre claro permite entender rápidamente qué hace una variable o función, mejora el mantenimiento del programa y reduce errores al trabajar en equipo.

···4. ¿Qué es la Programación Orientada a Objetos (POO)?

Es un paradigma de programación basado en objetos, los cuales representan entidades reales con características (atributos) y comportamientos (métodos). Permite organizar mejor el código y hacerlo más reutilizable y fácil de mantener.

···5. ¿Cuáles son los cuatro pilares de la Programación Orientada a Objetos?

Los cuatro pilares son:

- **Encapsulamiento:** proteger los datos y controlar su acceso.

- **Herencia:** permitir que una clase herede características de otra.

- **Polimorfismo:** un mismo método puede comportarse de diferentes maneras.

- **Abstracción:** mostrar solo la información necesaria y ocultar detalles internos.

···6. ¿Qué es la herencia en POO y cómo se utiliza en Java?

La herencia permite que una clase hija reutilice atributos y métodos de una clase padre. En Java se utiliza con la palabra clave extends. Esto evita repetir código y facilita la reutilización.

···7. ¿Qué son los modificadores de acceso y cuáles son los más comunes en Java?

Son palabras clave que controlan desde dónde se puede acceder a clases, atributos o métodos.

Los más comunes son:

- **public:** accesible desde cualquier parte del programa.

- **private:** solo accesible dentro de la misma clase.

- **protected:** accesible dentro del mismo paquete y clases hijas.

- **default (sin palabra clave):** accesible solo dentro del mismo paquete.

···8. ¿Qué es una variable de entorno y por qué es importante para Java o la programación en general?

Una variable de entorno es un valor configurado en el sistema operativo que los programas utilizan para funcionar correctamente. En Java, por ejemplo, se usa para indicar dónde está instalado el JDK mediante la variable JAVA_HOME. Son importantes porque permiten ejecutar programas y herramientas sin configurar rutas manualmente cada vez.
